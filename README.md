# Каталог автомобилей

Простая программа для поиска автомобилей по характеристикам.

## Как использовать

Программа ищет машины в каталоге по заданным параметрам.

**Пример:**
```python
# Найти все машины 2019 года выпуска
param = 'выпуск'
kriteriy = "2019"
```

**Результат:**
```
Бренд - BMW; Модель - 5 Series
Бренд - Volkswagen; Модель - Passat
```

## Доступные параметры для поиска

- `выпуск` - год выпуска
- `страна` - страна производитель
- `привод` - тип привода (задний, передний, полный)
- `двигатель` - объем двигателя
- `price` - цена
- `разгон до 100` - время разгона

## Примеры поиска

```python
# По стране
param = 'страна'
kriteriy = "Япония"

# По приводу
param = 'привод' 
kriteriy = "передний"

# По цене
param = 'price'
kriteriy = 1500
```

## Что делает программа

1. Создает каталог из 10 марок автомобилей
2. Добавляет параметр "рассрочка" для всех машин
3. Ищет машины по вашему запросу
4. Показывает найденные результаты

Просто запустите файл и программа найдет машины 2019 года выпуска. Чтобы искать по другим параметрам - измените значения в функции `main()`.
